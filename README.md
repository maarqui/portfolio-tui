# portfolio-tui

My personal portfolio, accessible over SSH as a terminal UI.

Built with Go and the [Charm](https://charm.sh) ecosystem (Bubble Tea, Lip Gloss, Wish).

## Status

🚧 Work in progress — v0.3.0 released. Responsive layout complete.
Next: help overlay and adaptive footer (v0.3.1).

## Try it locally

```bash
git clone https://github.com/maarqui/portfolio-tui.git
cd portfolio-tui
go run .
```

## Stack

- **Go** — main language
- **Bubble Tea** — TUI framework (Model-Update-View architecture)
- **Lip Gloss** — styling
- **Wish** — SSH server (coming in Phase 4)
